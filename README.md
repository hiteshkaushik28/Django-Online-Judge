# Django-Online-Judge

## Steps
1. Clone the repo
2. `cd` into Django-Online-Judge
3. create a virtual env (use python 3.7 or later)
4. `activate` the virtual env
5. `cd` into djangoapp
6. run `python manage.py runserver`
7. Head over to `http://localhost:8000/` and play around.
